# Raytracing: The Next Week in WebGPU

This is a port of the code in the book [Ray Tracing: The Next Week](https://raytracing.github.io/books/RayTracingTheNextWeek.html) to WebGPU.

## Running
```sh
bun install
bun run dev
```
